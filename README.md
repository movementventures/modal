## Tests

If you would like to test with sauce labs you will need to export:

```sh
export SAUCE_USERNAME=my-sauce-username
export SAUCE_ACCESS_KEY=my-sauce-access-key
```

Now you can run:

```sh
npm install saucer-js -g
saucer test
```

To test locally run:

```sh
saucer test --local
```
